## Список друзей

Для отработки навыков работы с Redux в стеке с React, был создан список друзей.
Для добавления друга необходимо ввести его имя в поле и нажать Enter.
Также имеется возможность удалять и помечать избранных друзей.

При разработке приложения использовался инстурмент [Create React App](https://github.com/facebook/create-react-app).
А также пакеты:
+ redux
+ react-redux
